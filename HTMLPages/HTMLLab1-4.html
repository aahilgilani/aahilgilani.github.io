Common HTML Tags

These are the most commonly used HTML tags and a short discription of each.

Block Elements

Block elements take up their own line and tale up the full width of the page.

Inline Elements

Inline elements only take up as much space as they need to. They do not start new lines>

a

This tag defines a hyper link.

Strong

Defines important text.

img

Defines an image.

html

Defines the root of a HTML document.
